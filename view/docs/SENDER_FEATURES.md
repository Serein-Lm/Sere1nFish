# Sender 输入框增强功能说明

## 功能概览

已将 PhishingPlatform 页面的 Sender 组件升级为功能完整的增强版输入框，包含以下特性：

## 新增功能

### 1. 📎 文件上传功能
- **拖拽上传**: 支持直接拖拽文件到输入框
- **点击上传**: 点击左侧附件按钮选择文件
- **粘贴上传**: 支持 Ctrl+V 粘贴文件（如截图）
- **附件预览**: 上传的文件会在 header 区域显示
- **数量限制**: 最多支持 5 个附件

### 2. 🎤 语音输入功能
- **语音识别**: 点击麦克风按钮开始录音
- **实时反馈**: 录音时显示动画效果
- **自动转文字**: 录音结束后自动转换为文本（当前为模拟实现）
- **状态提示**: 录音状态实时显示在底部

### 3. 📝 模板插入功能
- **快速插入**: 点击模板按钮插入预设文本
- **光标定位**: 文本插入到当前光标位置
- **可扩展**: 可以添加更多模板选项

### 4. 🧹 清空功能
- **一键清空**: 清空输入内容和所有附件
- **智能禁用**: 无内容时按钮自动禁用
- **确认提示**: 清空后显示提示信息

### 5. 📊 实时统计
- **字符计数**: 实时显示输入字符数
- **附件计数**: 显示已上传附件数量
- **录音状态**: 显示当前录音状态

### 6. ⚙️ 其他增强
- **自动高度**: 输入框自动调整高度（2-8 行）
- **取消请求**: 请求中可以取消操作
- **历史记录**: 预留历史记录功能入口
- **响应式设计**: 适配不同屏幕尺寸

## 使用方式

### 基础输入
```typescript
// 直接输入文本
输入你的需求 → 按 Enter 发送
```

### 文件上传
```typescript
// 方式 1: 点击附件按钮
点击 📎 图标 → 选择文件 → 自动上传

// 方式 2: 拖拽文件
拖拽文件到输入框 → 自动上传

// 方式 3: 粘贴文件
复制文件/截图 → Ctrl+V 粘贴 → 自动上传
```

### 语音输入
```typescript
// 点击麦克风按钮
点击 🎤 图标 → 开始录音 → 再次点击结束 → 自动转文字
```

### 模板插入
```typescript
// 点击模板按钮
点击 📄 图标 → 自动插入模板文本
```

### 清空内容
```typescript
// 点击清空按钮
点击 🧹 图标 → 清空所有内容和附件
```

# 审核规则

1. `page_kind` 必须按截图真实页面填写为文章详情、列表、视频、账号页、加载中或其他。
2. 提取当前页面可见的标题和账号；看不见时留空，不得从点击前候选复制。
3. `candidate_match` 表示当前页面与点击前候选是否为同一条内容。只有可见标题或账号明确吻合时才能给高分。
4. `target_match` 表示当前页面是否以目标主体为核心。同行业内容、多主体合集、仅偶然提及目标或相似名称不得给高分。
5. 当前可见标题和账号均未直接出现目标正式名或可靠别名时，`target_match` 必须低于 40。
6. 点击前候选摘要含有目标名称不能提高当前详情页的 `target_match`。

最终只返回调用方要求的结构化 Schema。

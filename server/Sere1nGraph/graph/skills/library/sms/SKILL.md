# 短信话术 Skill

## 话术阶段（script）

### 短信模板规则
- 字数控制在 70 字以内
- 包含可信的发送方标识
- 制造紧迫感但不过度

### 常用场景
1. 电话未接通后跟进："XX总您好，刚才电话可能信号不好断了，我是XX公司的张明，关于XX项目想和您沟通，方便时回电"
2. 邮件发送后提醒："XX总，刚给您发了一封关于XX项目的邮件，烦请查收，有问题随时联系我 138xxxx"
3. 微信好友申请辅助："XX总您好，我是XX公司张明，刚添加了您微信，有个XX项目想和您对接"

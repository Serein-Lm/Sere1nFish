# 招标投标场景案例

## 案例 1: 机场广告招标
**背景**: 目标为某机场集团采购部门
**伪造身份**: 某广告传媒公司商务经理
**逻辑链条**: 邮件发送投标意向书 → 电话确认收到 → 微信发送"标书模板"压缩包
**关键点**: 利用招标信息的公开性建立初始信任
**结果**: 成功 — 目标习惯性打开了"标书模板"

## 案例 2: IT 系统采购
**背景**: 目标为某政府单位信息中心
**伪造身份**: 某安全厂商售前工程师
**逻辑链条**: 邮件发送产品方案 → 电话跟进 → 发送"测试版安装包"
**关键点**: 利用政府单位对安全产品的刚需
**结果**: 成功 — 目标在内网环境运行了"安装包"

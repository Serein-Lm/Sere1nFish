# 招聘场景案例

## 案例 1: 猎头推荐
**背景**: 目标为互联网公司 HR
**伪造身份**: 某猎头公司顾问
**逻辑链条**: Boss直聘/脉脉联系 → 微信添加 → 发送"候选人简历汇总"压缩包
**关键点**: HR 对简历文件有天然的打开习惯
**结果**: 成功 — 目标直接解压查看了"简历"

## 案例 2: Offer 发放
**背景**: 目标为求职中的开发者（小红书暴露求职状态）
**伪造身份**: 某公司 HR
**逻辑链条**: 邮件发送 offer → 附带"入职材料模板"压缩包
**关键点**: 求职者对 offer 邮件的期待心理
**结果**: 成功 — 目标急切打开了"入职材料"

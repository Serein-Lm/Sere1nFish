## 关键词选择策略

1. **必选词**（所有公司都搜）：实习、内推、招聘
2. **推荐词**（大多数公司）：跳槽、面试、校招、工作
3. **可选词**（根据行业）：合作、供应商、加班
4. **行业词**（特定行业）：参考行业特定场景

## 搜索词组合规则

- 每个关键词 = `{公司口语名} + {场景词}`
- 优先使用口语化名称（如"字节"而非"字节跳动"）
- 每次搜索建议 3-5 个关键词，按优先级排序
- 高社工价值的场景词优先搜索
